Previous and other technical experience

I have been associated with Azure and C#/.NET environment for the last two, but I have experience (commercial and not) in many other technologies.

In my first professional job I wrote programs for microprocessors which control high power supplies. There I used C and real-time operation system from Texas Instruments. I developed .NET appliaction which gathered some data from processor and his peripherals.

In my next job I was responsible for cooperation and development applications related to SmartTV. There I used JavaScript and NativeClient (it was a solution from Google which enable to run native C++ code in web browser).

Then I migrate to Java stack (Java 8, Spring, Hibernate) and I prepared with the team proof of concept of the platform which was to facilitate contact of the personal trainers with their’s pupils - direct messages, preparing training plans, observing pupil’s progress and so on. Besides development backend part of application I was responsible for managing the backend developer team.

Off the clock I worked on battery management system to electric cart (both software and hardware sides). As a part of my BSc thesis I prepared end-to-end controller for electropneumatic positioner. Electropneumatic positioner is an element of automation system which control valve opening level regarding to state other parts of the system. I designed hardware, prepare software in C where I implementes two control alghoritms: PID and based on fuzzy logic and desktop software which send commands to hardware part of controller and gather data, so I have experience also in electronics and low-level programming.

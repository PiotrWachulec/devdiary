How to add permissions to hard drives on Ubuntu?

Sun, Aug 30, 2020 One-minute read

If you want to add privileges to external drive for writing and executing, you have to call below command:

sudo chmod ugo+wx <path_to_drive>

Of course you have to remember that giving all permissions is good probably for the home use only. In the other cases you always should give only minimum required permissions.
